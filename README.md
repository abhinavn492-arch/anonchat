# AnonChat v1 - May 8, 2026

## Local Setup
1. Install Node.js 18+ from nodejs.org
2. Open terminal in AnonChat folder
3. Run: npm install
4. Run: npm start
5. Open http://localhost:3000 in 2 browser tabs to test

## Deploy to Railway - Recommended
1. Push code to GitHub
2. railway.app → New Project → Deploy from GitHub repo
3. Add environment variables from.env file
4. Deploy. You get a URL like anonchat.up.railway.app
